연수 개요

주제: Notion을 활용한 스마트한 학급 운영 및 업무 효율화

대상

Notion을 처음 사용하거나 기본 기능만 알고 있는 초등학교 교사

학습 목표

  • Notion의 기본 인터페이스와 핵심 기능을 이해하고 활용할 수 있다.
  • 주요 단축키를 사용하여 Notion 작업 속도를 향상시킬 수 있다.
  • 교수학습 및 학급 운영에 필요한 개인 맞춤형 템플릿을 제작하고 활용할 수 있다.
  • Notion 데이터베이스와 '양식(Forms)' 기능을 활용하여 데이터를 효과적으로 수집하고 관리할 수 있다.
  • Notion의 고급 기능을 활용하여 학급 운영 및 관련 업무의 효율성을 극대화하는 실질적인 방법들을 습득한다.

준비물

  • 개인 노트북 (인터넷 연결 필수)
  • Notion 계정 (미리 가입 권장)

연수 전 준비사항

원활한 실습을 위해 미리 Notion 계정을 생성해 오시는 것을 권장합니다. 데스크톱 앱을 사용하실 분은 Notion 데스크톱 앱을 설치해 주세요.

도입 및 연수 안내

1

강사 소개 및 연수 목표 공유

본 연수는 여러분이 Notion을 자신 있게 사용하고, 학급 운영과 업무를 효율적으로 관리할 수 있도록 설계되었습니다.

Notion의 다양한 기능을 효과적으로 활용하여, 복잡하고 반복적인 학급 운영 및 관련 업무의 효율성을 극대화하는 방법을 배우게 됩니다.

2

Notion 소개: 왜 초등교사에게 유용한가?

  • Notion은 단순한 메모 앱을 넘어 학급 관리, 수업 준비, 자료 아카이빙, 협업 등 다양한 업무를 통합 관리할 수 있는 플랫폼입니다.
  • 산재된 정보(한글 파일, 엑셀 파일, 메모장 등)를 Notion 페이지 하나로 통합 관리할 수 있습니다.
  • 클라우드 기반으로 언제 어디서든 접근 가능하며, 자동 동기화됩니다.
  • Notion for Education 플랜을 통해 교사는 무료로 강력한 기능을 사용할 수 있습니다.

알아두세요!

교육자는 Notion for Education 플랜을 통해 프로 기능을 무료로 사용할 수 있습니다. 학교 이메일로 가입 후 Notion 교육자 프로그램에서 신청하세요.

Notion 기본 기능 및 인터페이스 탐색

실습 1: Notion 시작하기 및 첫 페이지 만들기

1

Notion 계정 확인하기

  • 웹 브라우저를 열고 www.notion.so로 이동합니다.
  • 이미 계정이 있다면 로그인합니다.
  • 계정이 없다면 "Sign up" 버튼을 클릭하고 이메일 주소를 입력하여 가입합니다.
  • 개인용(Personal) 계정을 선택합니다.
  • 로그인 후 Notion의 메인 화면이 표시됩니다.

참고하세요!

이 연수는 Notion 웹 버전 기준으로 진행되지만, 데스크톱 앱이나 모바일 앱에서도 동일하게 적용됩니다.

2

Notion 인터페이스 둘러보기

  • 왼쪽 사이드바: Notion의 메인 메뉴입니다.
    • 상단에는 워크스페이스 이름과 검색 아이콘이 있습니다.
    • 중간에는 자주 사용하는 페이지, 최근 페이지, 공유된 페이지가 표시됩니다.
    • 하단에는 설정, 업그레이드, 도움말 등의 메뉴가 있습니다.
  • 메인 편집 영역: 실제로 콘텐츠를 작성하고 편집하는 공간입니다.
  • 사이드바 하단의 + 새 페이지 버튼을 찾아봅니다.
3

첫 Notion 페이지 만들기

  1. 사이드바 하단의 + 새 페이지 버튼을 클릭합니다.
  2. 페이지 제목 영역에 "선생님 Notion 연습장"이라고 입력합니다.
  3. 아이콘 추가하기:
    • 제목 왼쪽의 회색 원 또는 "Add icon" 텍스트를 클릭합니다.
    • 이모지, 업로드, 아이콘, 랜덤 중에서 선택합니다.
    • 이모지를 선택한 경우, 원하는 이모지(예: 📝, 🎓, 🏫)를 클릭합니다.
  4. 커버 이미지 추가하기:
    • 제목 위 영역에서 "+ 커버 추가" 또는 "Add cover"를 클릭합니다.
    • 제공된 갤러리, 업로드, 링크 중에서 선택합니다.
    • 갤러리에서 마음에 드는 이미지를 선택합니다.
  5. 하위 페이지 만들기:
    • 페이지 본문 영역을 클릭하고 /page를 입력합니다.
    • "페이지" 또는 "Page" 옵션을 선택합니다.
    • 새로운 하위 페이지의 제목으로 "기본 블록 연습"이라고 입력합니다.
    • 같은 방법으로 또 다른 하위 페이지 "템플릿 활용"을 만듭니다.
  6. 페이지 구조 변경하기:
    • 왼쪽 사이드바에서 방금 생성한 하위 페이지 중 하나를 다른 페이지로 드래그합니다.
    • 드래그하여 들여쓰기나 순서를 변경할 수 있습니다.
    • 페이지를 다시 원래 위치로 드래그하여 돌려놓습니다.

교사 팁: 학급 운영에 필요한 다양한 페이지(학급 알림장, 주간 학습 계획, 학생 관리 등)를 만들어 체계적으로 관리하세요. 페이지 구조를 잘 정리하면 필요한 정보를 빠르게 찾을 수 있습니다.

핵심 블록 유형 및 단축키 활용 실습

실습 2: 다양한 블록 유형과 단축키 익히기

1

슬래시(/) 명령어 활용하기

Notion에서 가장 중요한 명령어는 슬래시(/) 명령어입니다. 이 명령어로 다양한 유형의 블록을 빠르게 생성할 수 있습니다.

  1. "기본 블록 연습" 페이지를 엽니다. (사이드바에서 클릭)
  2. 페이지 본문을 클릭하고 슬래시(`/`) 키를 누릅니다.
  3. 다양한 블록 옵션이 표시됩니다.
  4. 다음 블록을 차례로 만들어봅니다:
    • /text를 입력하고 선택 → "이것은 기본 텍스트 블록입니다."라고 입력
    • 새 줄에서 /h1을 입력하고 선택 → "제목 1" 입력
    • 새 줄에서 /h2를 입력하고 선택 → "제목 2" 입력
    • 새 줄에서 /h3을 입력하고 선택 → "제목 3" 입력
    • 새 줄에서 /bullet을 입력하고 선택 → "글머리 기호 목록 항목" 입력
    • 새 줄에서 /numbered를 입력하고 선택 → "번호 목록 항목" 입력
    • 새 줄에서 /toggle을 입력하고 선택 → "토글 목록 항목" 입력
    • 토글 블록의 화살표를 클릭하면 내용이 접히거나 펼쳐집니다.
    • 새 줄에서 /quote를 입력하고 선택 → "인용문 블록입니다." 입력
    • 새 줄에서 /callout을 입력하고 선택 → "중요한 정보를 표시하는 콜아웃입니다." 입력
    • 콜아웃 아이콘을 클릭하여 다른 이모지로 변경해봅니다.
    • 새 줄에서 /divider를 입력하고 선택하여 구분선 추가

시간 절약 팁!

자주 사용하는 블록 유형의 슬래시 명령어를 외워두면 작업 속도가 크게 향상됩니다. 예: /page, /table, /todo 등

2

단축키로 페이지 꾸미기

  1. 새 페이지를 생성하고 제목에 "단축키 연습"이라고 입력합니다.
  2. 제목 만들기:
    • 페이지 본문을 클릭하고 # 입력 후 스페이스바 → "오늘의 할 일" 입력
    • (Mac 사용자는 동일하게 # 입력 후 스페이스바)
  3. 콜아웃 블록 만들기:
    • 새 줄에서 /callout 입력 후 선택
    • "⭐ 중요 공지: 내일은 단축 수업입니다." 입력
    • 왼쪽의 ⭐ 아이콘을 클릭하여 다른 이모지로 변경해봅니다.
  4. 할 일 목록 만들기:
    • 새 줄에서 ` (백틱) 입력 후 스페이스바 → "수업 자료 준비" 입력
    • Enter 키를 눌러 새 항목 → "학생 상담" 입력
    • 다시 Enter 키를 눌러 새 항목 → "결재 서류 확인" 입력
    • 체크박스를 클릭하여 완료 표시를 해봅니다.
자주 사용하는 단축키 Windows Mac
굵게 Ctrl+B Cmd+B
기울임 Ctrl+I Cmd+I
밑줄 Ctrl+U Cmd+U
블록 복제 Ctrl+D Cmd+D
할 일 체크박스 [] + 스페이스 [] + 스페이스

교사 팁: 할 일 목록은 학급 업무 체크리스트나 학생 활동 체크리스트로 활용하면 편리합니다. 완료된 항목은 자동으로 구분되어 진행 상황을 한눈에 파악할 수 있습니다.

세션 4: 나만의 맞춤 템플릿 만들기 실습 (15분)

실습 3: 일일 학급 알림장 템플릿 만들기

1. 새 템플릿 페이지 생성하기 (2분)

  • 사이드바 하단의 + 새 페이지 버튼을 클릭합니다.
  • 페이지 제목에 "일일 학급 알림장 템플릿"이라고 입력합니다.
  • 아이콘 추가하기:
    • 제목 왼쪽의 회색 원을 클릭합니다.
    • 이모지 탭에서 학급 알림장에 적합한 이모지(예: 📢, 📝, 📆)를 선택합니다.
  • 커버 이미지 추가하기:
    • 제목 위 영역에서 "+ 커버 추가"를 클릭합니다.
    • 갤러리에서 밝고 명랑한 학교/교육 관련 이미지를 선택합니다.

2. 템플릿 기본 구조 만들기 (10분)

  • 알림 사항 섹션 만들기:
    • 페이지 본문을 클릭하고 `##` 입력 후 스페이스바 입력
    • "오늘의 알림 사항" 입력 후 Enter 키 누르기
  • 알림 사항 목록 추가하기:
    • `-` 입력 후 스페이스바 → "준비물:" 입력 후 Enter 키
    • `-` 자동 생성됨 → "숙제:" 입력 후 Enter 키
    • `-` 자동 생성됨 → "내일 일정:" 입력 후 Enter 키
    • `-` 자동 생성됨 → "특이사항:" 입력 후 Enter 키
  • 구분선 추가하기:
    • 새 줄에서 `/divider` 입력 후 선택하여 구분선 추가
  • 준비물 섹션 만들기:
    • 새 줄에서 `##` 입력 후 스페이스바 → "내일 준비물" 입력
  • 준비물 체크리스트 추가하기:
    • 새 줄에서 `` ` `` (백틱) 입력 후 스페이스바 → "색연필" 입력 후 Enter 키
    • `` ` `` 자동 생성됨 → "도화지" 입력 후 Enter 키
    • `` ` `` 자동 생성됨 → "실과 수업 재료" 입력 후 Enter 키
    • (샘플 준비물이므로 실제 사용 시 내용 변경 필요)
  • 학습 내용 섹션 만들기:
    • 새 줄에서 `/divider` 입력 후 선택하여 구분선 추가
    • 새 줄에서 `##` 입력 후 스페이스바 → "오늘 학습 내용" 입력
  • 학습 내용 테이블 추가하기:
    • 새 줄에서 `/table` 입력 후 선택
    • 3x3 테이블이 생성됩니다 (행과 열 조정 가능)
    • 첫 행에 "교시", "과목", "내용" 입력
    • 두 번째 행에 "1", "국어", "동시 쓰기" 입력
    • 세 번째 행에 "2", "수학", "도형의 넓이" 입력
    • 표 아래에 행을 추가하려면 표의 마지막 셀에서 Tab 키를 누릅니다.

3. 템플릿 활용하기 (3분)

  • 템플릿 복제 방법:
    • 사이드바에서 방금 만든 "일일 학급 알림장 템플릿" 페이지에 마우스를 올립니다.
    • 페이지 이름 오른쪽에 나타나는 "..." 메뉴 또는 마우스 오른쪽 버튼을 클릭합니다.
    • "복제" 또는 "Duplicate"를 선택합니다.
    • 복제된 페이지의 제목을 "5월 15일 학급 알림장"으로 변경합니다.
    • 내용을 필요에 맞게 수정할 수 있습니다.
  • 템플릿 버튼 만들기 (시간 여유가 있는 경우):
    • 원본 템플릿 페이지로 돌아갑니다.
    • 페이지 하단에 `/button` 입력 후 선택합니다.
    • "오늘 날짜로 알림장 생성" 텍스트를 입력합니다.
    • 버튼을 클릭하고 "새 페이지 추가" 액션을 선택합니다.
    • 페이지 위치를 지정하고, 제목을 "[날짜] 학급 알림장"으로 설정합니다.
    • "현재 콘텐츠 복사" 옵션을 선택한 후 "완료"를 클릭합니다.

세션 5: 데이터베이스 기초 및 양식(Forms) 기능 활용 실습 (20분)

실습 4: '학생 과제 제출' 데이터베이스 및 양식 만들기

1. 데이터베이스 생성하기 (5분)

  • 사이드바 하단의 + 새 페이지 버튼을 클릭합니다.
  • 페이지 제목에 "학생 과제 제출 현황"이라고 입력합니다.
  • 페이지 본문을 클릭하고 `/table` 또는 `/database`를 입력합니다.
  • "새 데이터베이스 생성" 또는 "테이블 뷰 인라인"을 선택합니다.
  • 테이블이 생성되면 기본적으로 "이름" 열이 생성됩니다.

2. 데이터베이스 속성(열) 설정하기 (7분)

  • 기본 속성 이름 변경하기:
    • "이름" 열 헤더를 클릭하고 "과제명"으로 변경합니다.
  • 새 속성 추가하기:
    • 테이블 오른쪽 끝의 "+" 버튼을 클릭하여 새 속성을 추가합니다.
  • 제출 학생 속성 추가:
    • "+" 클릭 → 속성 이름에 "제출 학생" 입력
    • 속성 유형에서 "텍스트" 선택
    • 오른쪽 상단의 "완료" 또는 "X" 버튼 클릭
  • 학년 속성 추가:
    • "+" 클릭 → 속성 이름에 "학년" 입력
    • 속성 유형에서 "선택" 선택
    • 옵션 추가 영역에 "1" 입력 후 Enter, "2" 입력 후 Enter... "6"까지 추가
    • "완료" 클릭
  • 반 속성 추가:
    • "+" 클릭 → 속성 이름에 "반" 입력
    • 속성 유형에서 "숫자" 선택
    • "완료" 클릭
  • 제출 파일/링크 속성 추가:
    • "+" 클릭 → 속성 이름에 "제출 파일/링크" 입력
    • 속성 유형에서 "파일 및 미디어" 또는 "URL" 선택
    • "완료" 클릭
  • 제출일 속성 추가:
    • "+" 클릭 → 속성 이름에 "제출일" 입력
    • 속성 유형에서 "날짜" 또는 "생성 일시" 선택
    • "완료" 클릭
  • 비고/코멘트 속성 추가:
    • "+" 클릭 → 속성 이름에 "비고/코멘트" 입력
    • 속성 유형에서 "텍스트" 선택
    • "완료" 클릭

3. 양식(Form) 뷰 생성하기 (5분)

  • 데이터베이스 상단의 "테이블" 텍스트 오른쪽에 있는 "+" 또는 "+ 보기 추가" 버튼을 클릭합니다.
  • 보기 유형 목록에서 "양식(Form)"을 선택하고 "생성" 버튼을 클릭합니다.
  • 양식 뷰 화면으로 전환됩니다.
  • 양식 제목 및 설명 편집:
    • 양식 상단의 "학생 과제 제출 현황" 텍스트를 클릭하여 "5월 15일 국어 과제 제출"과 같이 구체적인 제목으로 변경합니다.
    • 제목 아래의 설명 영역을 클릭하여 "기한: 5월 17일까지 제출해주세요. 기한 내 제출 시 가산점이 있습니다."와 같은 안내 문구를 추가합니다.
  • 양식 질문 순서 조정:
    • 각 질문(속성) 오른쪽의 6개의 점(핸들)을 드래그하여 순서를 변경합니다.
    • 권장 순서: 제출 학생 → 학년 → 반 → 과제명 → 제출 파일/링크 → 비고/코멘트 → 제출일
  • 필수 항목 설정:
    • 각 질문 오른쪽의 "•••" 메뉴를 클릭하여 설정을 엽니다.
    • "제출 학생", "학년", "반", "과제명"은 "필수"로 설정합니다.
    • "제출 파일/링크", "비고/코멘트"는 선택 사항으로 둡니다.

4. 양식 공유 및 응답 테스트하기 (3분)

  • 양식 뷰 오른쪽 상단의 "공유" 버튼을 클릭합니다.
  • "웹에 게시" 토글을 켭니다.
  • "양식 링크 복사" 버튼을 클릭하여 링크를 복사합니다.
  • 양식 테스트:
    • 복사한 링크를 새 탭에 붙여넣거나, 연수 참가자들과 공유합니다.
    • 학생이 되어 양식을 작성해봅니다:
      • 제출 학생: 자신의 이름 또는 가상의 학생 이름
      • 학년: 드롭다운에서 선택
      • 반: 숫자 입력
      • 과제명: "독서 감상문" 또는 다른 과제명
      • 제출 파일/링크: 파일 업로드 또는 URL 입력
      • 비고/코멘트: 추가 정보 입력
    • "제출" 버튼을 클릭합니다.
  • 응답 확인:
    • Notion으로 돌아가 데이터베이스 상단의 "양식" 텍스트를 클릭하고 "테이블"을 선택하여 테이블 뷰로 전환합니다.
    • 방금 제출한 응답이 새로운 행으로 추가되었는지 확인합니다.
    • 행을 클릭하면 세부 정보 페이지가 열립니다.

세션 6: 데이터베이스 고급 기능 및 자동화 맛보기 (15분)

실습 5: 관계형 데이터베이스 및 고급 기능 시연

1. 관계형 데이터베이스 구축하기 (5분)

  • 학급 명렬 데이터베이스 생성:
    • 새 페이지를 만들고 "2학년 3반 학생 명단"이라고 제목을 입력합니다.
    • `/table`을 입력하고 테이블 데이터베이스를 생성합니다.
    • 첫 번째 열의 이름을 "학생 이름"으로 변경합니다.
    • 새 속성을 추가합니다: "학번"(숫자), "성별"(선택: 남, 여), "연락처"(전화)
    • 몇 명의 가상 학생 정보를 입력합니다 (예: "김민준", "이서연", "박지훈")
  • 상담 기록 데이터베이스 생성:
    • 새 페이지를 만들고 "학생 상담 기록"이라고 제목을 입력합니다.
    • `/table`을 입력하고 테이블 데이터베이스를 생성합니다.
    • 첫 번째 열의 이름을 "상담 제목"으로 변경합니다.
    • 속성 추가: "상담 일자"(날짜), "상담 내용"(텍스트)
  • 관계형 속성 추가하기:
    • "학생 상담 기록" 데이터베이스에서 "+" 버튼을 클릭하여 새 속성을 추가합니다.
    • 속성 이름에 "학생"이라고 입력합니다.
    • 속성 유형에서 "관계형"을 선택합니다.
    • "관계 대상"에서 "2학년 3반 학생 명단"을 선택합니다.
    • "왕복"에서 "예"를 선택하고 "관계 이름"을 "상담 기록"으로 설정합니다.
    • "완료"를 클릭합니다.
  • 관계형 데이터베이스 사용해보기:
    • "학생 상담 기록" 데이터베이스에 새 항목을 추가합니다.
    • "상담 제목"에 "수학 과제 관련 상담"이라고 입력합니다.
    • "상담 일자"에 오늘 날짜를 선택합니다.
    • "상담 내용"에 "수학 과제 수행에 어려움을 겪어 개별 지도함"이라고 입력합니다.
    • "학생" 속성에서 앞서 추가한 학생 중 한 명을 선택합니다.
    • "학생 명단" 데이터베이스로 돌아가 선택한 학생을 확인하면, "상담 기록" 속성에 방금 작성한 상담 기록이 자동으로 연결된 것을 볼 수 있습니다.

2. 롤업 및 수식 기능 활용하기 (5분)

  • 과제 제출 현황 데이터베이스에 필드 추가:
    • "학생 과제 제출 현황" 데이터베이스로 돌아갑니다.
    • "+" 버튼을 클릭하여 "제출 여부" 속성을 추가하고 유형은 "체크박스"로 설정합니다.
    • "+" 버튼을 클릭하여 "점수" 속성을 추가하고 유형은 "숫자"로 설정합니다.
    • 몇 개의 샘플 과제를 더 추가하고 각각 제출 여부와 점수를 입력합니다.
  • 학생별 과제 제출률 계산을 위한 설정:
    • "2학년 3반 학생 명단" 데이터베이스로 돌아갑니다.
    • "+" 버튼을 클릭하여 "과제" 속성을 추가하고 유형은 "관계형"으로 설정합니다.
    • "관계 대상"에서 "학생 과제 제출 현황"을 선택합니다.
    • "왕복"에서 "예"를 선택하고 "관계 이름"을 "제출 학생"으로 설정합니다.
    • "학생 명단" 데이터베이스에 "+" 버튼을 클릭하여 "제출률" 속성을 추가합니다.
    • 유형은 "롤업"으로 선택합니다.
    • "관계"는 "과제"를 선택합니다.
    • "속성"은 "제출 여부"를 선택합니다.
    • "계산"은 "체크된 비율"을 선택합니다.
    • "완료"를 클릭합니다.
  • 평균 점수 계산:
    • "학생 명단" 데이터베이스에 "+" 버튼을 클릭하여 "평균 점수" 속성을 추가합니다.
    • 유형은 "롤업"으로 선택합니다.
    • "관계"는 "과제"를 선택합니다.
    • "속성"은 "점수"를 선택합니다.
    • "계산"은 "평균"을 선택합니다.
    • "완료"를 클릭합니다.

3. 워크플로우 자동화 및 협업 기능 시연 (5분)

  • 버튼 기능 활용:
    • "일일 학급 알림장 템플릿" 페이지로 돌아갑니다.
    • 페이지 하단에 `/button`을 입력하고 "오늘 알림장 생성" 텍스트를 입력합니다.
    • 버튼을 클릭하고 설정을 엽니다.
    • "새 페이지 추가"를 선택합니다.
    • "페이지 위치"에서 현재 워크스페이스 또는 원하는 폴더를 선택합니다.
    • "속성 편집"에서 "날짜"를 선택하고 `@오늘`을 입력합니다.
    • "제목"에 "[날짜] 학급 알림장"을 입력합니다.
    • "페이지 콘텐츠 추가"를 클릭하고 현재 페이지를 선택합니다.
    • "완료"를 클릭하고 버튼을 테스트합니다.
  • 동기화 블록 활용:
    • 새 페이지를 만들고 "중요 공지사항"이라고 제목을 입력합니다.
    • 내용으로 "⚠️ 다음 주 화요일은 학교 전체 소방 훈련이 있습니다. 간편한 복장으로 등교해주세요."라고 입력합니다.
    • 이 텍스트 블록을 선택하고 블록의 왼쪽 핸들을 클릭합니다.
    • "다음으로 전환" → "동기화 블록"을 선택합니다.
    • 블록이 붉은색 테두리로 표시됩니다.
    • "일일 학급 알림장 템플릿" 페이지로 이동합니다.
    • 페이지 어딘가를 클릭하고 `/sync`를 입력합니다.
    • "동기화 블록"을 선택하고 방금 만든 동기화 블록을 선택합니다.
    • "중요 공지사항" 페이지로 돌아가 내용을 수정해봅니다 (예: "복장" → "운동화").
    • "일일 학급 알림장 템플릿"으로 돌아가 내용이 자동으로 업데이트되었는지 확인합니다.
  • 협업 기능 시연:
    • 아무 페이지나 선택하고 상단의 "공유" 버튼을 클릭합니다.
    • "웹에 게시" 옵션과 권한 설정(보기/편집) 방법을 확인합니다.
    • 특정 텍스트를 선택하고 오른쪽 클릭 후 "댓글 추가"를 선택합니다.
    • "이 부분은 조금 더 자세히 작성해주세요."와 같은 댓글을 입력합니다.
    • `@` 기호를 입력하면 멘션 기능을 사용할 수 있다는 것을 안내합니다.
    • 페이지 오른쪽 상단의 시계 아이콘을 클릭하면 페이지 변경 이력을 확인할 수 있다는 것을 보여줍니다.

세션 7: 마무리 및 질의응답 (5분)

1. 교실 내 Notion 활용 아이디어 공유 (2분)

  • 학급 알림장/홈페이지:
    • 공지사항, 주간 학습 안내, 학급 행사 일정, 과제 안내, 사진 갤러리 등을 한 곳에서 관리
    • 학부모와 공유 가능한 페이지로 활용하여 가정 연계 강화
  • 수업 자료 아카이브:
    • 교과별/주제별 수업 자료, 참고 링크, 영상, 활동지 등을 체계적으로 정리
    • 데이터베이스를 활용하여 키워드 검색 및 필터링 기능 활용
  • 과제 관리 시스템:
    • 오늘 실습한 과제 제출 양식을 활용하여 학생들의 과제 제출 현황 관리
    • 제출 여부, 피드백, 평가 등을 한눈에 확인
  • 학생 관리:
    • 학생별 특성, 상담 기록, 학부모 면담 내용 등을 체계적으로 기록
    • 관계형 데이터베이스로 학생 정보와 다양한 활동 내역 연결
  • 프로젝트 기반 학습(PBL) 관리:
    • 프로젝트 개요, 일정, 자료 공유, 팀별 협업 공간 구성
    • 학생들에게 공유하여 협업 플랫폼으로 활용

2. 추가 학습 자료 안내 (1분)

  • Notion 공식 도움말 센터 (한글 지원)
  • Notion 한국 사용자 커뮤니티: 페이스북 그룹 '노션 한국 사용자 모임' 등
  • YouTube Notion 활용 강좌 채널: '노션 사용법', '노션 꿀팁', '노션 교육 활용' 등 검색
  • Notion 템플릿 갤러리
  • Notion for Education (교사 무료 플랜 신청)

3. 질의응답 및 마무리 (2분)

  • 연수 내용 관련 추가 질문을 받고 답변합니다.
  • Notion을 꾸준히 사용하며 자신만의 활용법을 찾아가도록 격려합니다.
  • 디지털 도구의 활용은 처음에는 학습 시간이 필요하지만, 익숙해지면 업무 효율성이 크게 향상된다는 점을 강조합니다.
  • 연수 참여에 감사 인사를 전하고 종료합니다.
